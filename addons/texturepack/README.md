# What is this
A script that changes textures.

# How to use

In this function add/replace the textures you want to change.
`"key": "url",`
Example:
```js
function loadPack(){
    return {
        "maps/tiles": "https://raw.githubusercontent.com/Neondertalec/tsmod/main/tiles5e12c370.jpg",
    }
}
```
